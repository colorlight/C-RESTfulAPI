函数说明

int32 RFDevice_Create ( RFDeviceConfig *  devConfig,
int8 *  deviceID 
)

新增设备。设备创建成功后，用户应保存返回的设备ID，当下一次上线时， 无需重新调用设备创建接口，直接使用RFDevice_Init 即可。 本接口包含RFDevice_Init 的功能。

参数
devInfo需要新增设备的设备信息
deviceID返回设备ID。缓冲区传入前需初始化
返回
参见OSAErrno.h
int32 RFDevice_Delete ( int8 *  APIKey)

从云端删除指定设备

参数
APIKey有权限进行删除操作的APIKey
返回
参见OSAErrno.h
void RFDevice_FreeDetails ( RFDeviceDetails *  details)

执行RFDevice_View 操作后，必须调用该接口释放空间

参数
details传入View接口的设备详细信息指针
int32 RFDevice_Init ( int8 *  deviceID,
int8 *  masterKey 
)

设备信息初始化，和RFDevice_Create 接口只需调用其中一个。需要向云端 添加设备时用Create接口，其它情况用Init接口

参数
deviceID设备创建时云端返回的ID
masterKey云端项目的MasterKey
返回
参考OSAErrno.h
int32 RFDevice_UpdateInfo ( RFDeviceConfig *  devConfig,
uint32  updateFlag 
)

更新设备信息

参数
devConfig需要更新设备的设备信息。
updateFlag指定需要更新的项，可同时更新多项内容。Flag指定的设备 信息项，需要在devConfig中正确填写
返回
参见OSAErrno.h
