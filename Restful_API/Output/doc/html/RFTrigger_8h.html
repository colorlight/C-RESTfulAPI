条件指定触发

触发参数为字符串类型的条件表达式。
$val[0]表示第一个数据流，$val[1]为第二个，以此类推 例如，第一个数据流上报数据点格式为：{“temperature”:22, “humidity”:56},若需设置温度大于30， 且湿度小于33时触发告警，则threshold中条件表达式可设置为：
“$val[0][‘temperature’] > 30 && $val[0][‘humidity’]< 33 ”
