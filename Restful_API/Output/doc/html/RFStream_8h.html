更新数据流信息

参数

APIKey有权限修改数据流信息的APIKey

参考OSAErrno.h
int32 RFStream_View ( int8 *  streamIDList,
int8 *  APIKey,
RFStreamInfo **  streamList 
)

查询数据流信息，只限本设备的数据流

参数
streamIDList指定需要查询的目标数据流ID，多个数据流以逗号隔开。为空指针 或长度为0则表示查询本设备的所有数据流
APIKey具有查询权限的APIKey
streamList返回RFStreamInfo类型的结构体数组指针，用户以“streamList[i]” 的形式查看各条数据流信息，使用后空间需释放
返回
大于等于0 - streamList中结构体成员个数 小于0 - 参考OSAErrno.h
