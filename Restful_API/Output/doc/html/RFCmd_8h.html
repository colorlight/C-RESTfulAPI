宏定义说明

#define RFCMD_STATUS_CREATED   1

命令已创建

#define RFCMD_STATUS_OFFLINE   0

目标设备不在线

#define RFCMD_STATUS_RESPRECV   4

目标设备正常响应

#define RFCMD_STATUS_RESPTIMEOUT   5

目标设备响应超时

#define RFCMD_STATUS_RESPTOOLARGE   6

目标设备响应过长

#define RFCMD_STATUS_SENDFAIL   3

命令发往目标设备失败

#define RFCMD_STATUS_SENT   2

命令已发往目标设备

int32 RFCmd_GetResponse ( int8 *  cmdUUID,
int8 *  respData,
uint32 *  respDataLen 
)

获取指定命令的响应内容，暂时只支持不超过4K的响应数据获取（包括HTTP头）

参数
cmdUUID指定需要查看响应的命令UUID
respData用于返回响应数据的缓冲区地址，传入前需初始化
respDataLen返回收到的响应数据长度
返回
